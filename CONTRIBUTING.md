# Contributing to go-winplugin

Thank you for contributing to go-winplugin.
This project aims to bring runtime plugin support to Windows for Go developers.

# Development Setup

```bash
git clone https://github.com/RamanSharma100/go-winplugin
cd go-winplugin

go test ./...
```

# Contribution Areas

- Runtime stability
- DLL loading improvements
- Cross-platform support
- Memory management
- Test coverage
- Documentation
- Performance optimization
- Reflection support
- Struct support

# Coding Standards

- Use clean Go formatting
- Prefer explicit error handling
- Avoid unnecessary abstractions
- Keep platform-specific code isolated
- Add tests for new functionality

# Pull Requests

Please ensure:

- Tests pass
- Code is formatted
- Documentation updated
- No breaking changes without discussion

# Reporting Issues

Please include:

- Go version
- Windows version
- MSYS2 version
- GCC version
- Reproduction steps
- Logs/output

# Roadmap Contributions

# Thank You

Every contribution helps improve Go plugin support on Windows.
